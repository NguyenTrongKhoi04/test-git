1) git init : tạo git trong folder để tương tác 
2) git remote add origin <link repo>: liên kết với link repo
3) git pull origin <nhánh code> : kéo code từ 1 nhánh về 
4) git add . : thêm full các file folder vào lưu trữ của git trên máy
5) git commit -m"<comment>": comment cho lệnh "git add" vừa rồi
6) git push origin <nhánh code>: đẩy lên nhánh code
7) git branch: hiển thị danh sách các nhánh
8) git checkout -b <new_branch_name>: tạo 1 nhánh mới
9) git checkout <branch_name>: chuyển sang nhánh cụ thể 
10) git merge <branch_name>: Hợp nhất (merge) một nhánh khác vào nhánh hiện tại.
